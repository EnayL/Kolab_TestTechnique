# Kolab_TestTechnique
Teste technique front-end pour paume 

## Les différentes pages : 

- /start.html : La page de lancement 
- /gestion.html : la page où on organise les rdv, travaux
- /kolab : page descriptif Kolab
- /kolab2 : page 2 descriptif Kolab
